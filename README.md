# Lumina Boutique

Premium boutique fashion for the modern muse.

## Project Structure

This project has been reorganized into a standard Vite/React structure for better maintainability and scalability.

### Directory Overview

- `src/components/`: Reusable UI components (Navbar, Footer, Hero, etc.).
- `src/pages/`: Top-level route components (Home, Collections, Admin).
- `src/assets/`: Static assets and images.
- `public/`: Public assets served directly.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Building for Production

Create an optimized production build:
```bash
npm run build
```

## Technologies Used

- **React**: Frontend library
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Styling
- **Framer Motion**: Animations
- **Lenis**: Smooth scrolling
- **React Router**: Routing
- **Lucide React**: Icons
